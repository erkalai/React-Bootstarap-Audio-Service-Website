## Run Locally

Clone the project

```CMD
  git clone https://github.com/erkalai/React-Bootstrap-DJ-Website.git
```

Go to the project directory

```CMD
  cd React-Bootstrap-DJ-Website
```

Install dependencies

```CMD
  npm install
```

Start the server

```CMD
  npm start
```
